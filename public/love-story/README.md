# Hướng dẫn thêm ảnh cho "Câu chuyện tình yêu"

## Cách upload ảnh:

1. **Chuẩn bị 4 ảnh** tương ứng với 4 cột mốc:
   - `1.jpg` - Lần đầu gặp nhau
   - `2.jpg` - Buổi hẹn hò đầu tiên  
   - `3.jpg` - Lời cầu hôn lãng mạn
   - `4.jpg` - Cùng nhau đến mãi mãi

2. **Đặt tên file** chính xác: `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`

3. **Copy ảnh vào thư mục này** (`public/love-story/`)

4. **Làm mới trình duyệt** (F5) để xem ảnh hiển thị

## Định dạng ảnh được hỗ trợ:
- `.jpg` / `.jpeg`
- `.png`
- `.webp`

## Kích thước ảnh đề xuất:
- Tỷ lệ: **1:1 (vuông)**
- Kích thước: **800x800 px** hoặc lớn hơn
- Dung lượng: dưới 2MB mỗi ảnh để load nhanh

## Lưu ý:
- Nếu file ảnh không tồn tại, sẽ hiện emoji thay thế
- Ảnh sẽ tự động cắt theo khung vuông (object-cover)
- Ảnh sẽ có hiệu ứng animation khi scroll đến
